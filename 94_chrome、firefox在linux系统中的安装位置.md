#  Chrome安装位置

`/opt/google/chrome`

在配置idea启动chrome时，直接选择`/opt/google/chrome/chrome`即可。

# Firefox安装位置

`usr/lib/firefox`

在配置idea启动firefox时，直接选择`usr/lib/firefox/firefox`即可。
